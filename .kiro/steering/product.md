# Vibe Notes

A minimalist, high-performance personal note-taking web application following the "Vibe Coding" philosophy.

## Core Value Proposition

- Fast, clean note-taking experience
- Secure by default with Row Level Security (RLS)
- Modern aesthetics with minimal friction

## Key Features

- User authentication (email/password)
- CRUD operations for notes
- Auto-save on blur
- Dashboard with note grid/list view
- Protected routes for authenticated users only

## Target Users

Individual users seeking a simple, secure place to capture thoughts and ideas.
